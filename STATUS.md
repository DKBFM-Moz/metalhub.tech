# 🏭 Проект: AI-калькулятор металлообработки
> Последнее обновление: 13 апреля 2026

---

## 🎯 Суть проекта

Веб-сервис: пользователь загружает чертёж (PDF/фото), указывает количество деталей и материал — AI сканирует чертёж и выдаёт расчёт стоимости изготовления. Монетизация через кредитную модель.

**Ключевая идея:** не раскрывать пользователю внутреннюю экономику цеха (ставки, накладные, маржу) — только итоговую цену с понятными факторами.

### Ползунки для пользователя (показываем)
- Материал (выбор из списка)
- Количество деталей
- Срочность (стандарт / срочно)
- Чей материал: наш / давальческий

### Скрытые параметры (только в панели владельца)
- Почасовая ставка станка
- Коэффициент накладных
- Целевая маржа
- Коэффициент риска/брака

### Монетизация
- 3 расчёта бесплатно при регистрации
- Пакет 10 кредитов — ~500–700 ₽
- Пакет 30 кредитов — ~1200 ₽
- Подписки пока нет — только кредиты
- Маржа на токенах: ~3–4x от реальной стоимости API

### Стоимость расчётов (ориентир)
| Тип | Стоимость API | Цена пользователю |
|-----|--------------|-------------------|
| Простая деталь | ~$0.02–0.05 | 1 кредит |
| Средняя | ~$0.05–0.10 | 2 кредита |
| Сложная | ~$0.10–0.20 | 3–5 кредитов |

### Инфраструктура
- **Хостинг:** Cloudflare Pages + Workers + D1 + R2
- **AI:** Anthropic API (Claude Sonnet)
- **Верификация:** телефон + почта (Twilio или SMS.ru)
- **БД:** Cloudflare D1 (SQLite)
- **Файлы:** Cloudflare R2

## 📐 Архитектура MVP (упрощённая)

```
Пользователь
    ↓
Cloudflare Pages (фронтенд — HTML/JS)
    ↓
Cloudflare Worker (бэкенд)
    ├── Проверка кредитов → D1
    ├── Сохранение файла → R2
    └── Вызов Anthropic API → расчёт
    ↓
Результат пользователю + списание кредита
```
