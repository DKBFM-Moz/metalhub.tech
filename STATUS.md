# 🏭 Проект: AI-калькулятор металлообработки (B2B)
> Последнее обновление: 16 апреля 2026

---

## 🎯 Суть проекта
Веб-сервис для производств: загрузка чертежа (PDF/фото) → AI-разбор →
интерактивный расчёт себестоимости и цены со всеми ползунками.
Целевая аудитория — технологи и экономисты цехов (B2B, не B2C).

---

## ✅ Что решено и реализовано

### Промпт AI-разбора чертежа (v1)
JSON-схема: part_info / geometry / operations / complexity.
Протестировано на реальном чертеже — Втулка цапфы 31-01-07-1299.00.015.
Confidence 95%, все параметры считаны корректно.

### Cost engine (v1) — реализован как интерактивный виджет
Формула: материал → мехобработка → накладные → маржа → цена без НДС → НДС 22% → цена с НДС.

**НДС: ставка 22%, НК РФ, действует с 2025 г. Ставка 20% упразднена.**
Все расчёты показывают: цена без НДС / сумма НДС / итог с НДС — обязательно.

Ползунки пользователя (открытые):
- Количество деталей, срочность, чей материал
- Цена стали (₽/кг), КИМ, ставка кузни
- Накладные расходы (%), рентабельность (%)
- Ставки станков (₽/ч): токарная, расточная, ТО, УЗК
- Нормы времени (мин/шт) по каждой операции

Сроки — в рабочих днях: поковка / мехобработка / ТО / ОТК.

### Стек
- Фронтенд: Cloudflare Pages (HTML/JS)
- Бэкенд: Cloudflare Worker
- AI: Anthropic API (claude-sonnet-4-20250514)
- БД: Cloudflare D1 · Файлы: R2
