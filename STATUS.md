# 🏭 Проект: MachCalc — AI-калькулятор металлообработки (B2B)
> Последнее обновление: 16 апреля 2026

---

## 🎯 Суть проекта

Калькулятор встроен в сайт **metalhub.tech** как страница `calculator.html`.

Сценарий "для даунов":
1. Загрузил чертежи (PDF/PNG/JPG, несколько файлов или drag & drop)
2. Написал задачу в текстовом поле (опционально)
3. Указал количество и срочность
4. Нажал — AI разобрал чертёж, подставил рыночные цены, выдал стоимость
5. При желании — развернул ползунки и скорректировал параметры вручную

**Целевая аудитория:** технологи и экономисты производств (B2B, не B2C).

JSON-схема: `part_info` / `geometry` / `operations` / `complexity`
Протестировано на чертеже Втулка цапфы 31-01-07-1299.00.015 — confidence 95%.

### Cost engine (v1)
```
материал + поковка
→ токарная + расточка + ТО + УЗК
→ × (1 + накладные%)
→ / (1 − маржа%)
→ × срочность
→ цена без НДС → + 22% НДС → итог с НДС
```
Сроки в рабочих днях: поковка / мехобработка / ТО / ОТК.

**НДС 22% · НК РФ · с 2025 г. Ставка 20% упразднена.**

Ползунки (все открыты пользователю):
- Количество, срочность, чей материал (свой / давальческий)
- Цена стали (₽/кг), КИМ, ставка кузни (₽/кг заг.)
- Накладные (%), рентабельность (%)
- Ставки станков (₽/ч): токарная карусель, расточная, ТО (₽/кг), УЗК (₽/шт)
- Нормы времени по каждой операции (мин/шт)

## 🔨 В работе — Парсинг цен на металл

### Почему прямой парсинг не работает
mcena.ru и большинство сайтов из списка рендерят цены через JS — голый HTML пустой.

### Решение — AI-поиск через Anthropic web_search tool
```
GET /api/prices?grade=40Х
  → проверяем KV кеш (TTL 24ч)
  → если нет → Anthropic API с web_search tool
  → AI ищет цены, усредняет из 3+ источников
  → { grade, price_per_kg, sources[], updated_at }
  → кешируем в KV на 24ч
```

### Ориентиры цен на апрель 2026 (из поиска)
| Марка | ₽/кг | Применение |
|-------|------|-----------|
| Ст3 / Ст20 | 55–65 | Общее машиностроение |
| 40Х | 65–85 | Валы, шестерни |
| 40ХН2МА | 120–160 | Ответственные детали |
| 09Г2С | 58–68 | Сварные конструкции |
| 12Х18Н10Т | 380–450 | Нержавейка |
| АД31 / АМГ | 180–220 | Алюминий |

### Что нужно добавить в wrangler.toml (дома)
```bash
wrangler kv namespace create PRICES_KV
# Вставить полученный id в wrangler.toml:
[[kv_namespaces]]
binding = "PRICES_KV"
id = "ПОЛУЧЕННЫЙ_ID"
```

## 📐 Полная архитектура

```
metalhub.tech (Cloudflare Pages)
    ↓ /api/* → Worker
Cloudflare Worker (machcalc-worker.kcoprofi.workers.dev)
    ├── POST /api/analyze   → Anthropic API → разбор чертежа
    ├── GET  /api/prices    → Anthropic web_search → цены  ← СТРОИМ
    └── GET  /api/health    → статус

Cloudflare KV (PRICES_KV)  → кеш цен TTL 24ч  ← НУЖНО СОЗДАТЬ
Cloudflare D1              → история расчётов  ← будущее
Cloudflare R2              → хранение чертежей ← будущее
```
